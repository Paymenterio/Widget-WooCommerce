== Description ==

Wtyczka integrująca bramkę płatniczą Paymenterio. Dzięki tej wtyczce możesz szybko i bezproblemowo przyjmować płatności za zakupy.

Z tym rozwiązaniem łatwo podłączysz w swoim sklepie metody płatności takie jak:

- BLIK
- przelewy natychmiastowe
- bankowe przelewy elektroniczne
- karty płatnicze
- przekazy pieniężne
- płatności PayPal
- płatności ratalne
- płatności Paysafecard

Dostawcą płatności jest Paymenterio Sp. z o.o.

== Installation ==

= WYMAGANIA INSTALACYJNE =

Poprawne korzystanie z modułu płatności Paymenterio jest możliwe tylko i wyłącznie w momencie posiadania konta oraz punktu płatności internetowej.

= INSTALACJA MODUŁU =

1.W panelu administracyjnym, przejdz do sekcji „Wtyczki”
1.1 Wybierz „Dodaj nową”
1.2 Kliknij przycisk „Wyślij wtyczkę na serwer”.
1.3 Wybierz plik archiwum.
1.4 Włącz wtyczkę.
2. Wybierz Ustawienia zainstalowanej wtyczki
3. W konfiguracji wtyczki Paymenterio uzupełnij wszystkie pola.
4. Uruchom wszystkie interesujące Cię metody płatności w ustawieniach Woocommerce.

= TESTOWANY =

Moduł płatności został przetestowany z wykorzystaniem WooCommerce w wersji 2.0 i nowszych.
